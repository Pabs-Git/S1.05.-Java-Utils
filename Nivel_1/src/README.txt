You will need to open your selected terminal in the cloned repository:

TO COMPILE ALL THE EXERCICES IN LINUX/MACOS:

. javac -d out src/*/**/*.java

TO COMPILE ALL THE EXERCICES IN POWERSHELL(WINDOWS):

. javac -d out (Get-ChildItem -Path src -Recurse -Filter *.java).FullName


Ejercicio 1:

	Execute:

-Linux/macOS: java -cp out Ejercicio_1.main.Main
-PowerShell(Windows): java -cp out Ejercicio_1.main.Main
